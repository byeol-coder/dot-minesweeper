# 숨은 점 찾기 · 게임 설계 요약

## 한 줄 소개
손끝으로 여러 점을 천천히 탐색해, 모양이 다른 ‘빛 점’을 찾고 별빛 나무를 깨우는 아동용 촉각 탐색 게임입니다.

## 스토리
밤바람이 별빛 나무의 빛 다섯 개를 숲속 돌 사이에 숨겼습니다. 플레이어는 안내 친구 도티와 함께 숲길을 지나며, 평범한 돌과 다른 촉각 패턴을 찾아 빛을 하나씩 되돌립니다.

## 실제 플레이 방식
닷패드는 손가락이 닿은 위치를 직접 감지하지 않으므로, 플레이어는 먼저 촉각으로 특별한 패턴을 찾고 방향키 또는 닷패드 기능키로 같은 위치까지 커서를 옮긴 뒤 확인합니다.

- 평범한 돌: 가운데 점 1개
- 숨은 빛: 십자 형태 5개 점
- 현재 커서: 선택 칸의 네 모서리 점이 깜빡임

## 단계
1. 반딧불 들판 · 2×2
2. 달빛 오솔길 · 3×2
3. 버섯 숲 · 3×3
4. 바람의 다리 · 4×3
5. 별빛 나무 언덕 · 5×4

시간 제한, 목숨, 게임 오버는 없습니다. 오답이 반복되면 방향 힌트에서 정확한 행·열 힌트로 단계적으로 강화됩니다.

## 닷게임즈 공용 튜토리얼
시작 안내 → 기기 확인 → 촉각판 읽기 → 조작 연습 → 핵심 규칙 → 질 수 없는 첫 판 → 혼자 하기의 7블록을 화면·촉각·음성 3채널로 제공합니다.

## 배포 전 필수 확인
- 호스트 루트에 `/tts.js` 존재
- `dotpad-sdk/DotPadSDK-3.0.0.js` 배치
- HTTPS의 Chrome 또는 Edge에서 실제 닷패드 BLE 연결 확인
- `?embed=1&preview=0`에서 헤더와 프리뷰 숨김 확인
- 한국어·영어 TW_TTS 재생 및 중복 음성 방지 확인
- F1~F4, 패닝키, 키보드 동등 조작 확인
